This project is a simple e-commerce web application developed as part of the Cloud Computing midterm exam. The application demonstrates the integration between a web server and a database hosted on AWS RDS.

The key focus areas of this project include:

  • Database integration with AWS RDS
  • Web application deployment on cloud infrastructure
  • Implementation of a complete web application with CRUD operations
  • Responsive and modern UI/UX design
Technologies Used:
Flask MySQL Bootstrap AWS RDS AWS EC2 HTML5 CSS3

Project Features

  • Display products from database
  • Add new products to the database
  • Responsive design using Bootstrap
  • Error handling and logging
  • Integration with AWS RDS database
  • Deployment on AWS cloud infrastructure
